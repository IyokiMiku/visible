# 一课一练试题编写规范（通用版）

> **适用范围**：所有省份、所有科目的一课一练试卷生成。本文档为出题的唯一权威标准，无需依赖外部记忆或上下文即可独立指导高质量试卷的生成。
>
> **核心理念**：以高职高考/高职分类考试真题标准为对标，确保每道题在知识准确性、选项均衡性、难度合理性、干扰项有效性四个维度均达到正式考试水平。
>
> **使用方式**：将本文档发送给任何 AI（ChatGPT/Claude/通义等），配合主题和题型要求即可直接生成符合标准的试卷。

---

## 零、对话模式完整出题流程

将本文档发给 AI 后，按以下格式发出指令即可：

```
请按照编写规范，为以下主题生成一份试卷：
- 省份：重庆市
- 考试类型：高职分类考试
- 教材：《机械基础》（高教版·第三版）
- 章节：第3节 材料与选用
- 主题：钢的热处理工艺
- 知识点范围：热处理目的与分类；退火/正火/淬火/回火工艺；表面热处理
- 题型：单选5+填空3+综合2
- 难度：80:10:10
```

### AI 生成流程（必须严格遵守）

```
第一步：确认题型数量（只生成指定的题型，未指定的不生成）
第二步：逐题生成，每出一题立即做自暴露检测：
        → 正确答案中≥4字关键词出现在题干中？是→废弃该题，重新出
第三步：出题时无需考虑答案分布，专注题目质量
第四步：全卷完毕后检查答案分布，若过于集中（如5题中4个A），
        通过交换选项顺序调整（把A选项和C选项文本互换，答案变为C）
第五步：执行质检清单（见第九-B节）
第六步：质检评分>90分→直接输出；有问题→只修复有问题的题（不重出全卷）；
        修复顺序为答案自暴露优先、其他单题问题其次、题干重复最后；最多修复3轮
第七步：输出最终试卷 + 简要质检报告
```

### 答案分布调整规则

出题时**不用**刻意控制答案分布，自然出题即可。生成完毕后检查：

| 选择题数量 | 触发调整的条件 | 调整方式 |
|-----------|--------------|---------|
| ≤5题 | 有4道及以上答案相同 | 选其中1-2题，将正确选项与另一选项的文本互换 |
| >5题 | 某字母占比超过50% | 同上，选部分题交换选项 |
| 其余情况 | 不调整 | 只在报告中告知分布情况 |

**交换方式示例**：第3题答案为A，交换A和C的选项文本后，答案变为C，题目内容不变。

### 质检后的处理规则

| 评分 | 处理 |
|------|------|
| >90分 | 自动通过，无需人工确认 |
| ≤90分且存在可定位问题 | 按优先级只修复对应题目：答案自暴露优先，其他单题问题其次，题干重复最后 |
| ≤90分且全卷性答案分布问题 | 先交换选项文本调整分布，避免重新出题 |
| 修复3轮仍不合格 | 报告问题并标记待人工审核或交由用户决定 |

### 真题风格参考使用规则

若提供“真题风格库”或历年真题样本，AI 只能模仿其命题口吻和题目结构，不得照搬内容：

| 可模仿 | 不可照搬 |
|--------|---------|
| 设问方式、题干长短、选项结构、干扰项来源、解析简洁程度 | 原题题干、选项文本、具体情境、年份、数值、答案组合 |

**知识准确性优先级**：当前考纲/主题 > 教材内容 > 编写规范 > 真题风格。真题风格库只解决“像不像真题”，不作为新知识依据。若风格样本与当前教材或考纲冲突，必须服从当前教材和考纲。

### 输出格式要求

| 要求 | 说明 |
|------|------|
| 从"一、单项选择题"直接开始 | 不要有任何标题、省份名、教材名等前缀 |
| 题型标题 | "一、单项选择题""二、填空题""三、综合题"等，用中文数字编号 |
| 选择题选项排列 | 短选项（≤18字）同行两个用制表符分隔；长选项（>18字）每个独占一行 |
| 填空题空格 | 用6个下划线 `______` 表示，禁止用括号（   ） |
| 判断题格式 | 题干末尾加（   ），答案为 √ 或 × |
| 答案格式 | 每题后紧跟 `【答案】X` 和 `【解析】完整因果句`；综合题或计算题含多个小问时，答案与解析按“（1）”“（2）”分行书写 |
| 公式算式 | 使用可直接排版的普通文本数学表达，如 `R=ρL/S`、`(18/12)×(3/2)`、`1.80 Ω`；禁止输出 LaTeX 包裹和命令，如 `\(...\)`、`$...$`、`\frac`、`\rho`、`\Omega`、`\times` |
| 禁止输出 | markdown标记、自检清单、额外标题、文件名、"以下是试卷"等 |

## 一、选项编写规范（最高优先级）

选择题选项质量是试卷专业度的第一标志。以下规则**强制执行**，任何违反即判为不合格题目。

### 1.1 选项长度均衡

**硬性规则：同一题中，最长选项字数 ÷ 最短选项字数 ≤ 2.0**

| 违规模式 | 典型表现 | 危害 |
|----------|---------|------|
| 一长三短 | 正确选项是完整描述句，错误选项仅1-3个字 | 学生凭长度直接选出答案 |
| 一短三长 | 三个干扰项堆砌细节，正确选项反而简洁 | 最短选项被猜为答案 |
| 详略失衡 | 正确选项专业术语密集，干扰项口语化 | 专业度差异暴露答案 |

### 1.2 干扰项实质性

**禁止使用以下类型的废选项：**

| 禁止的干扰项 | 问题 | 应替换为 |
|-------------|------|---------|
| "正常""无影响""无变化" | 不构成技术判断 | 该领域学生常见的错误认知 |
| "不确定""以上都不是" | 回避出题责任 | 具体的错误技术描述 |
| "更省油""更快""更好" | 无技术内涵 | 具体的错误因果关系 |
| "润滑""冷却""装饰" (与题目无关的随意词) | 一眼排除 | 与正确答案同层级的易混概念 |

**正确的干扰项来源：**
- 同章节容易混淆的相似概念（张冠李戴）
- 学生常见的理解偏差（因果倒置、主次混淆）
- 条件改变后的正确结论（条件缺失陷阱）
- 相关但不适用的技术方案（范围扩大/缩小）

### 1.3 选项句式结构一致

同一题的四个选项**必须**采用相同的语法结构：

| 错误（结构不一致） | 正确（结构一致） |
|-------------------|----------------|
| A.润滑 B.利用硅油产生阻尼吸收振动 C.冷却发动机 D.没什么用 | A.降低轴颈摩擦减少磨损 B.产生阻尼衰减扭转振动 C.吸收运转热量辅助散热 D.平衡离心力减小振动 |
| A.会拉缸 B.不影响 C.活塞环受热膨胀卡死 D.省油 | A.密封性提高燃烧更充分 B.环膨胀卡死导致拉缸 C.机油消耗量明显增大 D.发动机输出功率下降 |

### 1.4 选项连接词一致性

同一题四个选项的连接词使用必须保持结构对称，**禁止只有一个选项**含有“或”“和”“且”“以及”“并且”等连接词。

| 错误 | 问题 | 正确做法 |
|------|------|----------|
| A. 输入设备 B. 输出设备 C. 存储器和运算器 D. 控制器 | 只有 C 含“和”，结构明显特殊，容易暴露答案 | 四个选项都改为同类短语，或四个选项均采用“X和Y”结构 |
| A. 读取数据 B. 写入数据或删除数据 C. 传输数据 D. 保存数据 | 只有 B 含“或”，长度和结构突出 | 将选项统一为单一动作，或统一为并列动作 |

**原则**：如果一个选项需要使用连接词，其他选项也应使用同类并列结构；如果无法统一，应改写为不含连接词的同层级术语或短语。

### 1.5 答案位置分布

出题时不要为了预设 A/B/C/D 分布牺牲题目质量；每题首先保证答案唯一、选项合理、解析正确。生成后再检查全卷答案位置，若明显集中，则通过交换选项文本进行后处理调整。

| 选择题数量 | 触发调整的条件 | 调整方式 |
|-----------|--------------|---------|
| ≤5题 | 有4道及以上答案相同 | 选其中1-2题，将正确选项与另一选项的文本互换 |
| >5题 | 某字母占比超过50% | 选部分题交换选项，使分布更分散 |
| 其余情况 | 不调整 | 只在质检报告中告知分布情况 |

**严禁**为了凑答案分布而设置不合理干扰项、明显长短不一的选项或自暴露题干。

### 1.5 完整对照示例

**错误示范（第9练原题Q8）：**
> 8. 曲轴前端扭转减振器中硅油的作用（   ）
> A. 润滑　　B. 利用硅油的高粘度产生阻尼吸收扭转振动　　C. 冷却　　D. 装饰

问题：B选项19字，A/C/D仅2字；干扰项"装饰"荒谬可笑；答案100%可通过长度判断。

**正确写法：**
> 8. 曲轴前端扭转减振器中硅油的主要作用是（   ）
> A. 降低主轴颈与轴瓦间的摩擦系数　　B. 利用高粘度产生阻尼衰减扭转振动
> C. 带走曲轴高速旋转产生的摩擦热量　　D. 平衡曲轴旋转时产生的离心力

特点：四个选项均为"动词+对象+效果"结构，字数12-16字范围，干扰项均为发动机中真实存在的功能（分别对应轴承、减振器、冷却系、平衡重的功能）。

## 二、难度控制规范

### 2.1 难度配比

| 难度 | 占比 | 10题中 | 15题中 | 认知层次 |
|------|------|--------|--------|---------|
| 简单 | 80% | 8题 | 12题 | 识记——单一概念/定义的直接回忆 |
| 适中 | 10% | 1题 | 1-2题 | 理解+应用——需理解原理后判断情境结果，或简单代入计算 |
| 困难 | 10% | 1题 | 1-2题 | 综合分析——多步计算/故障推理链/跨知识点综合 |

**强制执行**：难度名称统一使用“简单、适中、困难”。每卷选择题中**必须包含**至少1道适中题和1道困难题，不得全部为简单概念回忆题。

**主观题困难题配置规则**：当一份试卷中主观题数量达到3道及以上时，应保证约1/3的主观题为“困难”难度。默认按“向下取整但至少1道”执行；若整套卷整体偏简单，可额外增加1道困难题。困难主观题应体现综合应用、情境分析、计算推导或方案说明，答案应有多个得分点，不得只是简单概念问答。

**题型内排序规则**：每个大类题型内部必须按难度由易到难排列，即“简单 → 适中 → 困难”。例如综合题部分有3道简单、2道适中、2道困难时，题号排序应为前3道简单，中间2道适中，最后2道困难。选择题、填空题、判断题、简答题、计算题、综合题等大类题型均按此规则执行。

### 2.2 各难度层级定义

| 难度 | 特征描述 | 出题手法 |
|------|---------|---------|
| 简单 | 教材中有明确唯一结论，题干与教材表述接近，无需推理即可作答 | 直接考定义、分类、名称、功能等 |
| 适中 | 需要理解概念后在新情境中应用，或进行1步简单计算/比较 | 给参数求结果（1步公式）；比较两个概念的异同；给定新情境判断适用的原理 |
| 困难 | 需要2步以上推理/计算，或综合多个知识点，或进行故障诊断推理链 | 多步计算题；“现象→原因→检修”推理链；跨章节知识的综合判断；反向推理（给结果推条件） |

**计算题难度特别规定**：单公式直接代入、只求一个数值结果的计算题，原则上只能判为“简单”。若计算题要判为“适中”，必须至少包含中间量、单位换算、公式选择、结果比较或简单工况判断中的一项。若计算题或含计算综合题要判为“困难”，不得只有一个公式和一个结果，必须体现多步骤计算、多公式组合、结果判断、工程解释、方案选择或改进措施中的至少两项，并形成多个评分点。

### 2.3 四维难度判定方法

试题难度应按“知识理解深度、计算复杂度、条件综合程度、工程判断要求”四个维度综合判定，不能只看题目字数或题型名称。

| 维度 | 简单 | 适中 | 困难 |
|------|------|------|------|
| 知识理解深度 | 直接识记教材概念、名称、分类、作用，通常只涉及1个知识点 | 需要理解原理后进行比较、判断或简单迁移，通常涉及1-2个相关知识点 | 需要综合多个知识点，解释原因、过程、影响或适用条件 |
| 计算复杂度 | 无计算，或只需直接识别标准数值 | 1步公式代入、简单单位换算或直接比较 | 2步及以上计算、先求中间量再判断结论，或计算后还需解释工程含义 |
| 条件综合程度 | 题干条件少，结论明确，基本无干扰信息 | 给出具体参数、场景或限制条件，需要筛选关键条件 | 条件较多且相互关联，需要先分析工况、对象、限制条件，再作答 |
| 工程判断要求 | 只问“是什么”“属于哪类”“主要作用” | 要求判断某方案、元件、工艺或现象是否适用，并说明基本原因 | 要求结合实际工况进行故障诊断、方案选择、安全判断、改进措施或风险分析 |

**判定规则**：
1. 一道题只要四个维度中大多数停留在“简单”，即判为简单。
2. 一道题若至少两个维度达到“适中”，且没有明显困难特征，判为适中。
3. 一道题若任一维度达到明显“困难”，并且答案需要分步骤说明或多个得分点，优先判为困难。
4. 主观题的困难难度必须体现至少两个困难特征，如“多知识点综合+工程判断”“情境分析+方案说明”“多步计算+结果判断”等。
5. 不能为了满足困难题比例而把简单概念题标成困难；困难题必须在题干条件、作答过程或评分要点上体现真实区分度。

### 2.4 计算/数值题要求

**每份练习中，选择题+填空题合计至少包含1-2道涉及数值计算或标准参数的题目**（视章节内容而定，纯概念章节可用参数记忆题替代）。

各学科常见可计算内容举例：

| 学科 | 可计算内容 |
|------|-----------|
| 发动机 | 压缩比ε=(Vc+Vh)/Vc、气缸工作容积Vh=πD²S/4、排量VL=Vh×i、配气相位角度、气门间隙 |
| 底盘 | 传动比计算、主减速器速比、轮胎滚动半径、制动距离 |
| 机械基础 | 齿轮传动比/分度圆直径d=mz、带传动速比、轮系传动比 |
| 机械加工 | 切削速度v=πdn/1000、转速、进给量、切削时间、尺寸公差 |
| 机械制图 | 比例换算、尺寸标注数值 |
| 电气设备 | 蓄电池容量、欧姆定律U=IR、功率P=UI |
| 计算机网络 | 子网划分、IP地址计算、传输速率换算 |
| 计算机组装 | 存储容量换算、总线带宽计算 |

**计算题编写规则：**
1. 给定明确数值条件，答案通过1-2步四则运算或公式代入可得
2. 选项中设置常见计算错误作为干扰项（如忘记乘以缸数、单位换算错误、公式记错等）
3. 填空题中可考查标准参数值的记忆（如"汽油机压缩比一般为____:1"）

## 三、题干书写规范

### 3.1 填空括号位置

括号 `（   ）` 位置应根据语义自然决定，**不强制放在句末**，优先放在语义最自然的位置：

| 题型 | 括号位置 | 示例 |
|------|---------|------|
| 概念/原因/定义类 | 句末 | "积屑瘤的形成原因是（   ）" |
| 功能/过程/填入类 | 句中自然位置 | "起动机将电能转化为（   ）驱动曲轴旋转" |
| 判断/选择类 | 句末 | "下列不属于切削液作用的是（   ）" |
| 条件/结果类 | 句中间 | "当活塞环开口间隙（   ）时，会导致拉缸" |
| 对象/部位类 | 句中间 | "飞轮安装在曲轴的（   ）" |

**原则**：括号位置应使题干读起来像一句自然流畅的话，选项填入后构成完整通顺的陈述句。

### 3.2 标点规范

所有内容**必须使用规范标点符号**，逗号（，）用于断句。

| 错误 | 正确 |
|------|------|
| "粗加工用大切削深度大进给量" | "粗加工用大的切削深度和进给量，以提高材料去除率" |
| "起动机将电能转化为机械能驱动曲轴" | "起动机将电能转化为机械能，驱动曲轴旋转" |

### 3.3 禁止事项

| 禁止 | 原因 | 正确做法 |
|------|------|---------|
| "某操作工""某零件""某厂" | 模糊，不专业 | 使用具体专业称谓或"一名维修技师""一台四缸发动机" |
| "请问""同学们""让我们" | 对话式/课堂用语 | 直接用陈述句出题 |
| "以上都是""以上都不是" | 偷懒出题 | 设置具体的第四个选项 |
| 题干包含正确答案关键词 | 自暴露 | 题干中≥4字与正确选项重复即违规，发现后立即重出该题 |
| 括号注解式写法，如"中央处理器（CPU）" | 题面像讲义注释，不像正式考试题；也会造成术语提示 | 直接使用教材中的规范名称，如"中央处理器"，不额外加英文缩写或括号解释 |

## 四、解析书写规范

### 4.1 禁用符号

解析中**严禁**使用以下符号，必须用完整文字替代：

| 禁用 | 错误示例 | 正确示例 |
|------|---------|---------|
| `→` | "积屑瘤→影响质量" | "积屑瘤会影响加工表面质量" |
| `=` | "全浮式=两端转动" | "全浮式活塞销两端均可自由转动" |
| `↑` `↓` | "温度↑→间隙↓" | "温度升高时间隙减小" |
| `+` | "密封+散热+刮油" | "气环具有密封、散热和刮油三个功能" |
| `=>` `≫` | 同上 | 用"导致""因此""会造成"等连接词 |

### 4.2 解析质量要求

- 解析必须是1-3句完整通顺的中文陈述句
- 必须解释**为什么正确答案是对的**（而非仅复述题干）
- **禁止单个短语式解析**：如"耐磨处理。""精密配合。"均不合格，必须写出完整因果逻辑
- 解析必须**直击题目要点**，与题干和正确答案形成因果闭环
- 建议简要说明主要干扰项为何错误
- 允许使用数学符号（π、≥、≤）和单位（mm、r/min、MPa）
- 计算题解析须列出计算步骤
- 含多个小问的综合题/计算题，答案与解析必须按小问换行书写，每行以“（1）”“（2）”等开头，避免多个小题挤在同一段
- 算式必须使用可直接排版的普通文本数学表达：乘法写作“×”，电阻率写“ρ”，欧姆写“Ω”，分式写成“L/S”或“(18/12)×(3/2)”；禁止输出 LaTeX 包裹和命令（如 `\(...\)`、`$...$`、`\frac`、`\rho`、`\Omega`、`\times`），避免进入 DOCX 后出现乱码

### 4.3 解析对照表

| 不合格解析（禁止） | 合格解析 |
|------------------|---------|
| "耐磨处理。" | "第一道气环承受最高温度和最大压力，镀铬或喷钼可显著提高其耐磨性和耐热性，延长使用寿命。" |
| "精密偶件不可互换。" | "喷油器针阀偶件配合间隙仅0.001~0.003mm，属于精密偶件，磨损后必须成对更换，不可与其他偶件互换。" |
| "连杆轴颈。" | "连杆大头孔内安装连杆轴瓦，通过轴瓦与曲轴连杆轴颈配合，将活塞的往复运动转化为曲轴的旋转运动。" |
| "前端=正时齿轮。" | "飞轮安装在曲轴后端法兰盘上，其主要作用是储存做功冲程的能量并输出扭矩。曲轴前端安装的是正时齿轮和扭转减振器。" |

## 五、多选题专项规范

1. **禁止全选**：每道多选题至少有1个干扰项
2. 正确选项数量控制在2-3个（总选项通常4个）
3. 各选项独立，不使用"以上全部""A和B""除C外都对"等表述
4. 多选题的选项同样遵守第一节的长度均衡、句式一致规则

## 六、判断题专项规范

1. **禁止直白复述**：判断题题干不能是教材原句的直接搬运，必须有变式改编
2. 错误判断题中的错误点应是学生容易忽略或混淆的细节，而非荒谬错误
3. 正确与错误的比例建议为 4:6 至 5:5（错误题略多，增加辨析难度）
4. 解析中必须明确指出错误点在哪里以及正确说法是什么

## 七、情景化出题规范

**情景化题目占比不低于30%。**

情景化 = 题目包含具体的工程场景、工况条件、故障现象、操作步骤或技术参数，而非直接问"XX的定义是什么"。

| 非情景化（禁止超过70%） | 情景化（至少30%） |
|------------------------|-----------------|
| "发动机冷却系统的作用是（ ）" | "某发动机水温表指示超过100℃且水箱溢水，最可能的故障原因是（ ）" |
| "压缩比的定义是（ ）" | "某四缸发动机缸径82mm冲程75mm，其单缸工作容积约为（ ）" |
| "继电器的功能是（ ）" | "一辆车在行驶中突然熄火，检查发现点火线圈无高压电输出，应首先检查（ ）" |

## 八、质检流程（强化版）

每卷生成后**立即**执行本地质检。质检结果驱动以下处理逻辑：

| 质检结果 | 处理方式 |
|---------|---------|
| 有严重问题（选项失衡/废选项/解析过短/题干重复等） | **只修复有问题的题目**，最多修复3轮；不整卷重生 |
| 答案自暴露 | **最高优先级修复**，只重出涉及题目 |
| 全卷性答案分布问题 | 先尝试交换选项文本修复，避免重新出题 |
| 专业正确性严重问题（错答/答案不唯一/解析冲突/计算错误/条件不足/安全表述错误） | 必须定向修复对应题目，修复后重新核验，不得直接发布 |
| 无问题 | 直接保存 DOCX |
| 修复3轮仍不合格 | 保留最高分版本，并标记“待人工审核” |

**重要**：质检报告仅在终端/对话框中展示，**不写入试卷文档**。质检的作用是驱动定向修复，优先保留已合格题目，而非仅产出一份报告。

具体检查维度如下：

### 第一道：选项质量质检（针对问题1）

逐题检查以下项目：

| 检查项 | 判定标准 | 不合格处理 |
|--------|---------|-----------|
| 长度比 | max(选项字数)/min(选项字数) ≤ 2.0 | 重写选项使长度接近 |
| 干扰项实质性 | 无"正常""无影响""不动"等废词 | 替换为真实易混概念 |
| 句式一致性 | 四选项语法结构相同 | 统一为同一句式 |
| 答案分布 | 统计ABCD各几题，任一选项占比不超过40% | 调整部分题目的选项顺序 |
| 无自暴露 | 题干无≥4字与正确选项重复 | 废弃该题重新出题 |

### 第二道：难度与题型质检（针对问题2、3）

| 检查项 | 判定标准 | 不合格处理 |
|--------|---------|-----------|
| 难度名称 | 全卷只使用“简单、适中、困难”三档，不使用“容易、中等、难题”等旧称 | 统一替换为“简单、适中、困难” |
| 难度分布 | 必须有≥1题适中+≥1题困难 | 将部分简单题升级 |
| 主观题困难比例 | 主观题数量≥3时，约1/3主观题应为困难，默认向下取整但至少1道 | 将部分主观题改为综合应用、情境分析、计算推导或方案说明题 |
| 题型内难度排序 | 每个大类题型内部按“简单→适中→困难”排序 | 调整同一题型内部题号顺序，不改变题目内容 |
| 四维难度判定 | 难度应按知识理解深度、计算复杂度、条件综合程度、工程判断要求综合判定 | 重新判定难度；不得把简单概念题硬标为困难 |
| 计算/数值覆盖 | 选择题+填空题中≥1题涉及数值 | 补充计算题 |
| 计算题深度 | 单公式直接代入只能判为简单；适中计算题至少有中间量、单位换算、结果比较或简单判断；困难计算题至少有2个计算/判断步骤或多个评分点 | 重写计算题，使其包含“算+判”或“算+判+析/改” |
| 含计算综合题关联性 | 综合题中的计算小问应服务于后续判断、分析或措施，不能是互不关联的小问堆砌 | 重构小问逻辑，形成“计算→判断→分析/改进”链条 |
| 情景化比例 | ≥30%题目含具体场景/参数/工况 | 将概念题改写为情境题 |
| 判断题对错比 | 错误题占40%-60% | 调整对错分布 |

### 第三道：文字格式质检

| 检查项 | 判定标准 |
|--------|---------|
| 编号连续 | 全卷题号1,2,3...连续无跳号 |
| 答案完整 | 每题都有【答案】和【解析】 |
| 禁用符号 | 解析中无→↑↓=+=>等符号 |
| 标点规范 | 无缺标点的长句 |
| 答案格式 | 选择题答案为大写字母；判断题为√或× |

### 第四道：内容范围合规质检

| 检查项 | 判定标准 |
|--------|---------|
| 考纲覆盖 | 全部题目在考纲规定的知识范围内 |
| 不超纲 | 无考纲未覆盖的超纲内容 |
| 无重复 | 同卷任意两题不考查相同知识点 |
| 答案正确 | 逐题验证答案的正确性（特别是计算题） |

### 第五道：专业正确性质检（借鉴正式试卷审核标准）

专业正确性质检用于发现“形式合格但不能发布”的硬伤。以下问题一经发现，均按严重问题处理，必须定向修复后重新核验。

| 检查项 | 判定标准 | 不合格表现 | 处理方式 |
|--------|---------|------------|----------|
| 答案唯一性 | 单项选择题在题干条件下只能有一个正确答案；主观题评分点明确 | 两个及以上选项均可成立；题干缺少“主要、通常、最适合、在某条件下”等限定 | 补充限定条件、重写选项或改题 |
| 答案解析一致性 | 【答案】与【解析】支持的结论完全一致 | 答案标A但解析支持B；判断题答案与解析正反相反；多小问答案与解析错位 | 同步修正答案、解析和选项 |
| 计算复核 | 公式、代入、单位换算、计算结果和最终结论均正确 | 公式用错、单位漏换、数值算错、结果与选项不一致、计算后工程判断错误 | 重新计算并修正答案解析 |
| 条件充分性 | 题干给出的条件足以推出答案，解析不引入题干未给出的结构、参数或工况 | 答案依赖图但无图；解析自行增加零件、参数、载荷方向、工况假设 | 补充题干条件、增加图示，或改为可由题干推出的答案 |
| 专业概念准确性 | 概念、原理、因果关系和适用范围符合教材、考纲及工程常识 | 概念混淆、因果倒置、把局部结论扩大为普遍结论、把“通常”写成“必然” | 改写为有条件、准确、可验证的表述 |
| 工程安全性 | 涉及机械、电气、液压、气压、维修、检修、急停和操作安全时，安全逻辑必须可靠 | 将危险操作写成正确措施；断电/断气/急停后错误认为所有机构必然安全；忽略泄压、锁止、防护、互锁等 | 按安全操作和工程风险重新设计答案 |
| 难度与大纲匹配 | 难度符合考纲层级和一课一练定位；超纲内容仅作为困难题或拓展题 | “了解”层级出复杂综合计算；整卷都偏难或都偏简单；拓展内容当作必考核心 | 调整题目难度或标注为拓展/困难 |
| 缺图缺表 | 题干引用图、表、零件位置或回路时，必须提供对应信息 | 出现“如图所示”“根据下图”但无图；要求识读结构却没有结构描述 | 补图、补表或改为纯文字可作答题 |

**高风险模块重点复核**：对滚动轴承、液压气动、齿轮/轮系计算、公差配合、机械制图投影与标注、电路计算、故障诊断、安全操作等模块，应额外核验公式、标准术语、条件完整性和工程安全逻辑。

**常见严重问题示例**：
1. 单选题中两个选项都正确，例如题干未限定工况导致多种连接方式均可实现同一功能。
2. 解析第一句支持B选项，但【答案】写成A。
3. 计算题公式正确但幂次、单位换算或近似值计算错误，导致最终答案错误。
4. 题干只给出部分结构，答案却写出题干未出现的套筒、挡圈、阀件或其他零件。
5. 安全题写成“急停后一键断气即可保证所有气缸回到安全位置”等绝对化结论。

### 第六道：重复度检测

#### A. 同卷自重复检测

同一份试卷内，任意两道题之间必须保持差异化：

| 检查项 | 判定标准 | 严重程度 |
|--------|---------|---------|
| 题干相似度 | 全题型题干均参与查重；去除模板词后，连续短语、核心关键词或去模板题干高度相似即判重复 | 严重 |
| 知识点不重叠 | 不得有两题考查完全相同的知识点 | 严重 |
| 选项不复用 | 不得有两题的正确答案文本完全相同 | 警告 |
| 解析不雷同 | 不得有两题解析的核心表述完全一致 | 警告 |

**相似度计算方法**：先提取全题型题干（选择题、判断题、填空题、综合题均参与），去除“下列、关于、正确的是、错误的是、说法、作用”等出题模板词，再综合判断：①是否存在较长连续相同片段；②核心关键词片段是否高度重合；③去模板后的题干字符集合是否高度相似。满足任一高风险条件即判为题干重复。

#### B. 编号连续主题跨练查重检测（仅限 xxx（一）/xxx（二）/xxx（三））

只有主题名称明确写成 `xxx（一）`、`xxx（二）`、`xxx（三）` 这类连续编号时，才需要做跨练差异化。普通不同主题之间不互相查重，例如 `电阻定律` 不需要与 `欧姆定律` 查重，`电阻定律部分` 也不会因为前一个不同主题而注入避重要求。

| 检查项 | 判定标准 | 严重程度 |
|--------|---------|---------|
| 题目重复率 | 相邻编号练习之间题干相似度 > 60% 的题对数量 ≤ 总题数的 20% | 严重 |
| 知识点互补 | 相邻编号练习覆盖角度不得完全相同，每练应有一定独占考查角度 | 严重 |
| 难度差异化 | 相邻编号练习的困难题不得考查完全相同的子知识点 | 警告 |
| 题型结构一致 | 同一连续主题内各练的题型配置和数量通常保持一致 | 建议 |

**连续主题差异化设计原则**：

1. **只约束同基础主题的编号续篇**：`电阻定律（二）` 参考 `电阻定律（一）`，但不参考 `欧姆定律`。
2. **角度变换**：同一知识点可出现在相邻编号练习中，但必须从不同角度命题（如（一）考定义，（二）考应用场景或计算）。
3. **禁止简单换序**：不得仅通过调换选项顺序或替换数值来制造“不同”的题目。

**检测时机**：
- 脚本模式（`create.py`）：生成 `xxx（二）` 时参考 `xxx（一）` 的题目摘要，生成 `xxx（三）` 时参考 `xxx（二）` 的题目摘要。
- 对话模式：AI 生成编号续篇时，必须回顾前一编号练习内容并自检差异度。

## 九、逐题自检清单（出题时每题必过）

每道题输出前，按以下10项逐一自查：

```
□ 1. 考纲对标：题目知识点在考纲范围内
□ 2. 难度定位：明确本题属于简单/适中/困难哪一级，并按知识理解深度、计算复杂度、条件综合程度、工程判断要求判断
□ 3. 情景化：是否为情景题（若本卷情景化不足30%则本题应为情景题）
□ 4. 选项长度比：max/min ≤ 2.0
□ 5. 选项实质性：无废选项，干扰项均为真实易混概念
□ 6. 选项句式：四选项语法结构一致
□ 7. 答案位置：不为凑分布牺牲题目质量；全卷完成后再统一检查并通过交换选项调整
□ 8. 陷阱设计：至少1个干扰项包含具体陷阱类型
□ 9. 无自暴露：题干不含正确答案关键词≥4字（若有则立即废弃该题重出）
□ 10. 答案正确：逻辑验证无误
□ 11. 解析质量：非单词短语，而是完整因果句，直击题目要点
```

## 九-B、对话模式质检流程（AI 在 Cursor/ChatGPT 等对话中生成时必须遵守）

当 AI 在对话（而非脚本）中生成试卷时，**必须**在生成后立即执行以下质检流程。质检结果在对话中报告给用户，**绝不写入试卷文本/文档**。

### 流程

```
第一步：生成完整试卷文本
第二步：对自己的输出执行以下质检项
第三步：根据质检结果决定处理方式
```

### 质检项（逐一执行）

| # | 检查项 | 判定标准 | 严重程度 |
|---|--------|---------|---------|
| 1 | 选项长度比 | 每题 max÷min ≤ 2.0 | 严重 |
| 2 | 废选项 | 无"正常""无影响""不动""任意"等 | 严重 |
| 3 | 答案分布 | ABCD 各占比不超过 40%，无集中 | 严重 |
| 4 | 选项句式一致 | 同题四选项语法结构相同 | 严重 |
| 5 | 难度分布 | ≥1题适中 + ≥1题困难 | 警告 |
| 6 | 解析质量 | 无单短语解析，均为完整因果句 | 严重 |
| 7 | 禁用符号 | 解析中无 → ↑ ↓ = + => | 警告 |
| 8 | 答案自暴露 | 题干无≥4字与正确选项重复（发现则重出该题） | 警告 |
| 9 | 情景化比例 | ≥30%题目含具体场景 | 警告 |
| 10 | 格式正确 | 无markdown标记/无额外标题/无自检清单 | 严重 |
| 11 | 同卷无自重复 | 任意两题题干相似度 ≤ 50%，无相同知识点重复考查 | 严重 |
| 12 | 双卷查重（仅限显式连续编号主题） | 仅 `xxx（一）`、`xxx（二）`、`xxx（三）` 之间需要跨练差异化；普通不同主题之间不互相查重 | 严重 |

### 处理逻辑

| 质检结果 | AI 应执行的操作 |
|---------|----------------|
| **评分>90** | 直接输出试卷，附简要报告："质检通过，答案分布 A:X B:X C:X D:X，评分 XX/100" |
| **有单题严重问题或答案自暴露** | 按优先级只重新出有问题的题目：答案自暴露优先，其次其他单题问题，最后处理题干重复；保留其他题，修复后重新质检 |
| **全卷性答案分布问题** | 先尝试交换选项文本修复，避免重新出题 |
| **仅有轻微警告** | 报告问题，询问用户是否保留 |
| **修复后仍有问题**（第3轮） | 报告问题，询问用户是否接受或标记待人工审核 |

### 报告格式（在对话中展示）

```
📋 质检报告
- 评分：XX/100
- 答案分布：A:X B:X C:X D:X
- 严重问题：X 个 [列出]
- 警告：X 个 [列出]
→ 处理：[定向修复 / 询问用户 / 通过]
```

### 重要约束

1. **质检报告只在对话中展示**，绝不写入试卷输出文本
2. **自检清单**只在 AI 内部执行，绝不输出给用户看到的文本中
3. 修复时无需用户重新输入指令，AI 基于上一版试卷只重出有问题的题目，其他题保持不变
4. 最多修复 3 轮，之后报告问题让用户决定或标记待人工审核

## 十、简答题与计算题专项模板

本节用于规范非选择类主观题的生成方式。简答题重在“要点完整、表述规范、层次清楚”；计算题重在“条件明确、步骤完整、单位正确、结果可信”。

### 10.1 简答题适用范围

简答题适用于考查概念理解、作用说明、步骤归纳、原因分析、区别比较和故障处理思路等内容。简答题不得写成单纯填空题，也不得写成开放性作文题。

| 适合出简答题 | 不适合出简答题 |
|--------------|----------------|
| 简述某机构的作用、组成或工作过程 | 只问一个术语名称 |
| 说明某工艺的操作步骤或注意事项 | 答案只有一个数字或符号 |
| 分析某故障现象的可能原因 | 无唯一评分要点的开放讨论 |
| 比较两个易混概念的区别 | 可直接改成单选且更清楚的问题 |

### 10.2 简答题题干模板

简答题题干应明确答题任务，常用“简述”“说明”“分析”“比较”“列举”等动词，避免“谈谈你的看法”“你认为怎样”等开放式表述。

| 类型 | 推荐题干模板 | 示例 |
|------|--------------|------|
| 作用类 | 简述……的主要作用。 | 简述离合器的主要作用。 |
| 组成类 | 简述……由哪些主要部分组成。 | 简述带传动装置由哪些主要部分组成。 |
| 步骤类 | 简述……的基本操作步骤。 | 简述普通车床对刀的基本操作步骤。 |
| 原因类 | 分析……产生的主要原因。 | 分析发动机水温过高的主要原因。 |
| 区别类 | 比较……与……的主要区别。 | 比较退火与正火的主要区别。 |
| 措施类 | 说明……时应采取的处理措施。 | 说明发现制动跑偏时应采取的检查措施。 |

**题干要求：**
1. 必须限定答题范围，避免答案无限扩展。
2. 若要求列举数量，应写明“写出2点”“至少写出3项”等。
3. 情景化简答题必须给出具体工况、故障现象或操作背景。
4. 不得在题干中直接暴露答案要点的完整关键词。

### 10.3 简答题答案模板

简答题答案应按要点分条书写，每个要点必须是完整句或完整短句，不得只堆砌关键词。通常控制在2-4点，除非题干明确要求更多。

**标准模板：**

```
X. 简述……。
【答案】（1）……。
（2）……。
（3）……。
【解析】本题考查……。答题时应从……、……和……等方面作答，核心是说明……。
```

**答案书写规则：**
1. 每个要点单独成行，以“（1）”“（2）”“（3）”编号。
2. 每个要点只写一个核心意思，避免一条中混入多个无关内容。
3. 要点顺序应符合专业逻辑，如“组成→作用”“现象→原因→措施”“公式条件→计算过程→结论”。
4. 简答题【解析】不重复抄答案，应说明答题思路、知识依据或易错点。
5. 若题干问“区别”，答案应使用对应结构，逐项比较，不得只分别介绍两个概念。

### 10.4 简答题合格示例

```
12. 简述离合器的主要作用。
【答案】（1）保证汽车平稳起步，使发动机动力逐渐传递给传动系。
（2）便于换挡时暂时切断动力，减小齿轮冲击。
（3）防止传动系过载，在冲击载荷过大时起到一定保护作用。
【解析】离合器位于发动机与变速器之间，核心作用是控制动力传递的接合与分离。答题时应围绕起步、换挡和过载保护三个典型工况展开。
```

### 10.5 简答题禁止写法

| 禁止写法 | 问题 | 正确做法 |
|----------|------|----------|
| “作用：起步、换挡、保护。” | 只有关键词，不是完整答案 | 写成完整作用句 |
| “答案很多，合理即可。” | 无评分标准 | 给出明确要点 |
| “谈谈对热处理的理解。” | 过于开放 | 改为“简述钢热处理的目的” |
| “离合器有哪些作用和结构？” | 一题多问且范围过大 | 拆成作用题和组成题 |
| 解析只写“见答案” | 没有解释思路 | 简述知识依据或易错点 |

### 10.6 计算题适用范围

计算题适用于考查公式理解、单位换算、参数代入、传动比计算、功率电阻计算、公差尺寸计算等内容。计算题可以出现在选择题、填空题、综合题或独立计算题中，但主观计算题必须体现完整步骤。

计算题不得长期停留在“单公式直接代入”的低层级。带计算的题目应尽量从“求一个结果”升级为“计算结果服务于判断、分析或方案选择”。

| 适合出计算题 | 不适合出计算题 |
|--------------|----------------|
| 已知参数完整，可用教材公式求解 | 缺少关键条件，需自行假设 |
| 1-3步可完成的基础计算 | 超出教材和考纲的大量推导 |
| 需要单位换算或结果判断 | 只考死记硬背的术语 |
| 可设置常见错误作为干扰项 | 结果不唯一或依赖经验估算 |

### 10.6.1 计算题深度分级

| 难度 | 最低要求 | 典型表现 |
|------|----------|----------|
| 简单 | 1个公式，直接代入，求出单一结果 | 已知齿数求传动比；已知电压电流求电阻；已知模数齿数求分度圆直径 |
| 适中 | 在直接计算基础上，至少加入1个中间量、单位换算、公式选择、结果比较或简单判断 | 先求分度圆直径再求传动比；计算后判断增速/减速；计算后与给定要求比较 |
| 困难 | 至少包含2个计算/判断步骤，或包含多公式组合、结果判断、工程解释、方案选择、故障分析、改进措施中的至少2项 | 计算关键参数后判断是否满足工况；比较两个方案并说明选择理由；根据测量数据判断故障原因；计算后提出调整措施 |

**硬性要求**：
1. 单公式直接代入的计算题不得标为困难。
2. 标为适中的计算题，不能只写“代入公式得……”，必须有比较、判断或中间量。
3. 标为困难的计算题或含计算综合题，答案至少应包含3个评分点，并至少出现2个计算/判断步骤。
4. 困难计算题应优先设计为“计算+判断+分析”或“计算+比较+方案选择”，而不是增加无意义的大数运算。
5. 如果一份试卷只有1道计算题，该题原则上不应只是简单计算，至少应达到适中水平；若有2道及以上计算题，应尽量包含1道适中或困难计算题。

### 10.6.2 “算—判—析—改”设计思路

适中和困难计算题优先采用“算—判—析—改”的结构：

| 环节 | 含义 | 示例任务 |
|------|------|----------|
| 算 | 根据公式计算关键参数 | 求传动比、转速、分度圆直径、电流、功率、公差等 |
| 判 | 将计算结果与要求、标准或工况比较 | 判断是否满足转速要求、尺寸限制、功率要求或安全条件 |
| 析 | 解释计算结果的工程含义或产生原因 | 说明为什么是减速传动、为什么负载过大会导致异常、为什么实际值低于理论值 |
| 改 | 若不满足要求，提出调整、检修或改进措施 | 调整齿数/模数、增大包角、检查阀件、增加保护装置等 |

简单计算题可以只包含“算”；适中计算题至少包含“算+判”；困难计算题应尽量包含“算+判+析”或“算+判+改”。

### 10.6.3 含计算综合题设计模板

综合题中的计算小问不能孤立存在，前一问的计算结果应服务于后一问的判断、分析或措施。

| 模板 | 结构 | 适用场景 |
|------|------|----------|
| 参数计算+是否满足要求+改进措施 | （1）计算关键参数；（2）判断是否满足要求；（3）若不满足，说明调整措施 | 机械传动、液压气动、电工电子、机械加工 |
| 故障现象+计算验证+原因分析 | （1）根据测量数据计算关键量；（2）判断是否异常；（3）分析可能原因 | 汽车维修、电气控制、设备检修 |
| 方案比较+计算+选择理由 | （1）分别计算两个方案的关键指标；（2）比较结果；（3）选择方案并说明理由 | 传动方案、加工参数、材料/工艺选择 |
| 工程限制+计算+安全判断 | （1）计算理论值；（2）与限制值比较；（3）说明是否安全或是否需要保护措施 | 液压气动、电气安全、设备操作 |

禁止把综合题写成互不关联的小问堆砌。例如“（1）求一个数；（2）写一个概念”若二者没有逻辑关系，不应作为高质量综合题。

### 10.7 计算题题干模板

计算题题干必须包含“已知条件、求解目标、单位要求”三要素。所有数值必须有单位，必要时说明取值精度。

| 类型 | 推荐题干模板 | 示例 |
|------|--------------|------|
| 公式代入类 | 已知……为……，……为……，求……。 | 已知电阻两端电压为12 V，通过电流为2 A，求电阻值。 |
| 单位换算类 | 已知……，按……换算，求……。 | 已知主轴转速为600 r/min，工件直径为50 mm，求切削速度。 |
| 多步计算类 | 已知……，先求……，再求……。 | 已知齿轮齿数和模数，先求分度圆直径，再求传动比。 |
| 情景应用类 | 某设备……，测得……，判断/计算……。 | 某电路测得电压为24 V、电阻为6 Ω，计算电流大小。 |

**题干要求：**
1. 已知条件必须完整，不让学生凭经验补条件。
2. 单位必须统一或明确要求换算。
3. 若需取近似值，应说明“结果保留一位小数”或“π取3.14”。
4. 不得设置繁琐无意义的大数计算，重点应是公式选择和专业理解。
5. 纯概念章节不强行设置计算题，可改为标准参数记忆题或数值判断题。

### 10.8 计算题答案与解析模板

计算题【答案】应直接给出最终结果，必须带单位；【解析】必须按“公式—代入—计算—结论”的顺序书写。算式使用普通文本数学表达，禁止 LaTeX 命令。

**标准模板：**

```
X. 已知……，求……。
【答案】……
【解析】根据公式……。代入数据得……。计算结果为……，因此……。
```

**多小问模板：**

```
X. 已知……，求：（1）……；（2）……。
【答案】（1）……
（2）……
【解析】（1）根据公式……，代入数据得……，所以……。
（2）根据……，代入数据得……，所以……。
```

**解析书写规则：**
1. 必须写出所用公式或计算依据。
2. 必须写出关键代入过程，不能只给最终结果。
3. 最终结果必须带单位；无单位的比值应说明“传动比为……”或“压缩比为……”。
4. 若有单位换算，必须在解析中体现换算过程。
5. 若结果需要四舍五入，应写明近似结果。
6. 公式写作普通文本，如 `U=IR`、`d=mz`、`v=πdn/1000`、`P=UI`，禁止写成 `\frac`、`\times`、`\Omega` 等 LaTeX 命令。

### 10.9 计算题合格示例

```
13. 已知某电阻两端电压为24 V，通过电流为3 A，求该电阻的阻值。
【答案】8 Ω
【解析】根据欧姆定律U=IR，可得R=U/I。代入数据得R=24/3=8 Ω，因此该电阻的阻值为8 Ω。
```

```
14. 一对外啮合标准直齿圆柱齿轮，主动轮齿数为20，从动轮齿数为60，求该齿轮传动的传动比。
【答案】3
【解析】齿轮传动比等于从动轮齿数与主动轮齿数之比。代入数据得i=60/20=3，因此该齿轮传动的传动比为3。
```

### 10.10 计算题禁止写法

| 禁止写法 | 问题 | 正确做法 |
|----------|------|----------|
| “答案：8” | 缺少单位 | 写“8 Ω” |
| “R=24/3=8” | 解析过短，缺少依据 | 先说明欧姆定律，再代入计算 |
| 使用 `\frac{24}{3}` | DOCX 转换易乱码 | 写作 `24/3` |
| 题干只写“求电阻” | 条件不完整 | 给出电压、电流及单位 |
| 多小问挤在一段 | 阅读和排版困难 | 按“（1）”“（2）”分行 |
| 计算结果与解析单位不一致 | 专业错误 | 全题统一单位并复核 |

### 10.11 简答题与计算题质检项

| 检查项 | 简答题判定标准 | 计算题判定标准 |
|--------|----------------|----------------|
| 题干明确 | 任务动词清楚，答题范围明确 | 已知条件、求解目标、单位完整 |
| 答案完整 | 2-4个要点，分条书写 | 最终结果正确且带单位 |
| 解析有效 | 说明答题思路或知识依据 | 有公式、代入、计算和结论 |
| 格式规范 | 使用“（1）”“（2）”编号 | 多小问按小问分行 |
| 难度合理 | 不做开放作文题 | 1-3步计算为主，不超纲 |
| 无自暴露 | 题干不直接列出答案要点 | 题干不直接给出可照抄的结果 |

## 十一、DOCX 格式规范

- **编写说明**：蓝色虚线单格表格（#4472C4，1.5pt），楷体10.5pt加粗
- **标题**：三行（省份 一课一练 / 教材名 出版社·版次 / 章节 练号 主题），宋体14pt加粗居中
- **页边距**：上下2.54cm，左右3.18cm
- **分隔图**：编写说明与标题之间，左对齐，4.60cm宽
- **中英文/公式混排**：中文用指定字体，拉丁字符用 Times New Roman；公式优先使用普通文本数学表达（如 `R=ρL/S`、`1.80 Ω`），避免在源文本中保留 LaTeX 命令
- **出版社参数**：`add_editorial_note()` 支持 `publisher` 参数（默认"高教版"），机工版传 `"机工版"`
- **选项对齐**：选择题选项使用 `add_question_options_table()` 输出为无边框表格，确保列对齐
- **答案/解析颜色**：【答案】和【解析】的标签及正文均使用红色字体 (RGB: 255,0,0)
- **大类题型标题**：如"一、单项选择题""二、填空题"使用黑体12pt**加粗**，不添加任何 markdown 标记

### 选项布局规则

- 任一选项 > 18字（含"A. "前缀）→ 单列排列（每选项独占一行）
- 同排两选项长度比 > 1.6 → 单列排列
- 其余 → 双列排列（同行两个选项，用制表符分隔）
- 无选项文本（判断题等）→ 普通段落

### AI 输出文本格式要求（输入给代码处理前）

AI 生成的原始试卷文本**必须**遵守以下规则，否则代码转 DOCX 时会产生格式错误：

| 要求 | 说明 |
|------|------|
| 禁止 markdown 标题 | 不得使用 `#`、`##`、`###` 等标记，直接从"一、单项选择题"开始 |
| 大类标题纯文本 | "一、单项选择题""二、填空题""三、综合题"等直接书写，无前缀标记 |
| 题型名称 | 单选用"单项选择题"，多选用"多项选择题"，禁止笼统写"选择题" |
| 选项排列 | 短选项（≤18字）同行两个用制表符分隔；长选项（>18字）每个独占一行 |
| 填空格式 | 用6个下划线 `______` 表示空格，禁止用括号（   ） |
| 禁止自检清单 | 自检清单仅供 AI 内部质检使用，严禁写入输出文本 |
| 禁止额外标题/文件名 | 输出开头不得包含文档标题、省份标题、版本说明等 |

## 十一、文件命名规范

```
第x练 {主题} {省份}（高职分类考试/高职分类考试）《教材名》（{出版社}·{版次}）一课一练（解析版）.docx
```

**省级行政区名称要求**：文件名、文档标题和编写说明中的省份/地区名称必须使用规范全称。自治区不得简写，统一写作：
- `内蒙古自治区`，不得写作 `内蒙古`
- `新疆维吾尔自治区`，不得写作 `新疆`
- `西藏自治区`，不得写作 `西藏`
- `广西壮族自治区`，不得写作 `广西`
- `宁夏回族自治区`，不得写作 `宁夏`

**版次格式要求**：出版社简称 + 中间圆点 + 中文版次，例如：
- ✓ `高教版·第三版`
- ✓ `机工版·第二版`
- ✗ `高教第3版`（错误：缺少"版"字和中间圆点，使用阿拉伯数字）
- ✗ `第3版`（错误：缺少出版社简称）

不含识图/作图题型。

## 附录A：各陷阱类型说明

| 陷阱类型 | 定义 | 出题手法 |
|----------|------|---------|
| 偷换概念 | 用相似但不同的概念替代正确概念 | "分度圆"改为"基圆"、"传动轴"改为"驱动轴" |
| 主次混淆 | 将次要功能/原因说成主要的 | "散热是活塞环的主要功能"（实际是密封） |
| 因果倒置 | 颠倒原因和结果的关系 | "拉缸导致活塞环卡死"（实际是环卡死导致拉缸） |
| 范围扩大 | 将局部结论扩大为普遍结论 | "所有发动机都用水冷"（风冷也存在） |
| 范围缩小 | 将普遍结论缩小为个别情况 | "只有柴油机有压缩比"（汽油机也有） |
| 绝对化表述 | 使用"一定""必须""所有""任何" | "配气相位角一定相同"（不同机型不同） |
| 条件缺失 | 省略关键前提条件 | "压缩比越大效率越高"（不提爆震限制） |
| 张冠李戴 | 将A零件的功能/特征说成B零件的 | "飞轮的作用是衰减扭转振动"（那是减振器） |

## 附录B：答案分布后处理速查表

生成选择题时不必预先套用固定答案序列，优先保证题目质量、答案唯一和选项合理。全卷生成后再按以下规则检查并调整：

| 选择题数量 | 触发调整的条件 | 调整方式 |
|-----------|--------------|---------|
| ≤5题 | 有4道及以上答案相同 | 选其中1-2题，将正确选项与另一选项的文本互换，并同步修改【答案】字母 |
| >5题 | 某字母占比超过50% | 选部分高频答案题交换到低频字母，使分布更分散 |
| 其余情况 | 不调整 | 只在质检报告中提示分布情况 |

**注意**：答案分布调整只能交换选项文本和答案字母，不能为了分布改坏题干、干扰项或解析。
