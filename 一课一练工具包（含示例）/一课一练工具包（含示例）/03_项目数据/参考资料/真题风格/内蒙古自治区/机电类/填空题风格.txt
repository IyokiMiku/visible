【填空题风格】

一、总体定位

内蒙古机电类填空题稳定为 20 小题，总分通常 80 分。近年常按“每空 2 分”设置，多数小题含 2 个或多个空。题目以基础术语、分类、性质、公式直接应用、标准代号和简单参数关系为主，强调答案的准确性和教材规范表达。

二、设问口吻

1. 多采用陈述句挖空形式，不设置复杂情境。
2. 常见句式：
   - “______按______可分为______和______。”
   - “______是反映______与______关系的曲线。”
   - “在______电路中，______在相位上______于______。”
   - “当______时，______为______。”
   - “已知______，则______为______。”
   - “某标记中，______表示______。”
3. 语言平实，空前空后提示明确，答案通常短小。
4. 同一小题多个空多为并列、对应或递进关系。

三、题干长度

1. 单题一般 20—70 字。
2. 计算填空可给 2—5 个参数，但计算步骤应控制在 1—2 步。
3. 多空题可稍长，但每个空应可独立判断或按公式顺序求出。
4. 不宜让前一空错误导致后续所有空无法作答。

四、答案形式

1. 标准术语：发射极、集电极、基极；编码器、译码器；退火、正火、回火等。
2. 分类名称：甲类、乙类、甲乙类；同步、异步；串行、并行等。
3. 关系词：增大、减小、不变、超前、滞后、相等、为零。
4. 数值结果：V、A、Ω、mm、Hz、rad/s、r/min、角度等，必要时带单位。
5. 成组答案：两个物理量、两个分类、两个变化趋势、两个方程数量等。

五、常考内容

（一）电类
- 二极管、三极管结构、材料、输入/输出特性曲线；
- 功率放大分类、整流滤波稳压组成、理想运放特性；
- 正弦交流电解析式、有效值、频率、初相位；
- 电感、电容相位关系，RLC 谐振，感抗、容抗；
- 电源模型、支路电流法方程数量、万用表使用；
- 逻辑门、触发器、寄存器、计数器、编码器、译码器、BCD 码。

（二）机械与材料
- 滑动/滚动轴承、花键、平键、齿轮、四杆机构；
- 齿轮参数、机构极位夹角、传动比；
- 公差、上/下极限偏差、极限尺寸；
- 钢的退火、正火、淬火、回火、调质；
- 有色金属、钢铁材料分类、铸铁与钢性能。

六、失分点设计

1. 相近术语混淆：发射极/集电极、编码/译码、同步/异步、串联/并联。
2. 方向关系混淆：电压电流超前/滞后，线量/相量关系。
3. 公式代入错误：角频率、容抗、感抗、功率、传动比、公差。
4. 单位遗漏或换算错误。
5. 标准代号含义不清。
6. 热处理工艺冷却方式或目的混淆。

七、难度控制

1. 以基础和中等偏易为主。
2. 概念填空答案必须唯一，不宜开放。
3. 计算结果数值应整洁，避免繁琐小数。
4. 可按“电类基础→数字电路→机械材料”的顺序排列，形成章节顺序感。

八、解析表达

1. 概念题：说明依据哪个定义或分类标准。
2. 关系题：说明由哪个性质或公式得到。
3. 计算题：列核心公式，代入并给出单位。
4. 多空题按空依次说明，避免长篇扩展。

九、保守模板

1. “______按______不同，可分为______和______。”
2. “______是反映______与______之间关系的曲线。”
3. “在______电路中，______在相位上______于______。”
4. “已知______为……，______为……，则______为______。”
5. “某标准件/机构/零件的标记为……，其中______表示______。”
6. “生产中把______和______相结合的热处理称为______。”
7. “用______法分析电路时，需列出______个______方程和______个______方程。”
