# 素材：一课一练试卷生成工具包

## 1. 项目概述
- 一课一练试卷生成工具包是一套面向高职/高职分类考试的AI驱动试卷批量生产系统
- 核心目标：根据考纲、教材目录、真题风格和考点规划表，批量生成解析版/原卷版Word试卷
- 定位：分层控制的试卷生产流程，不是简单"把考点丢给AI出题"
- 已覆盖省份：重庆市、四川省、内蒙古自治区
- 已覆盖考类：机械加工类、汽车类、电气技术类、电子技术类、土建类、机电类
- 支持多本教材独立拆分，每本教材生成独立规划表和试卷系列

## 2. 项目背景
- 职业教育分类考试需要大量针对性的"一课一练"练习卷
- 传统人工出题效率低、质量参差不齐，难以规模化
- AI生成试卷面临质量不可控、风格不统一、格式不规范等挑战
- 本项目通过多层控制体系（规划表→编写规范→题型定义→真题风格→质检修复），实现可控的AI试卷生产
- 项目从重庆机械加工类起步，逐步扩展到多个省份和考类

## 3. 核心数据
- 代码模块数：10个生成器模块 + 规划表/OCR/质检等多个独立工具集
- 工具脚本总数：56个Python脚本
- 已有输出成果：323个Word文档、224个原始文本、133个zip包
- 规划表覆盖：多省多考类多教材
- 支持题型：单选、多选、判断、填空、简答、计算、综合等
- 质检维度：答案自暴露、题干重复、选项失衡、解析过短、答案分布集中
- OCR方案：PyMuPDF + Tesseract + RapidOCR三套方案
- 真题风格支持：按省份/考类/题型维度蒸馏命题风格

## 4. 关键架构
- 分层控制体系：教材目录定主题 → 考纲定知识点边界 → 规划表定题量题型难度 → 编写规范定质量底线 → 题型定义定个性化画像 → 真题风格定设问方式
- 生成器模块化：runner(主流程) + config_io(配置) + planning(规划表解析) + prompts(提示词) + references(参考资料) + text_generation(文本生成) + text_processing(文本清洗) + quality(质检修复) + docx_generation(Word生成) + postprocess(后处理)
- 教材优先模式：有教材目录时先读目录定主题，再匹配考纲知识点；无教材才退回到仅考纲模式
- 真题风格只迁移风格不照搬内容

## 5. 项目亮点
- 教材目录OCR驱动：自动扫描教材目录页，PyMuPDF+Tesseract识别，结构化输出，与考纲自动匹配
- 真题风格蒸馏：从历年真题中抽取命题风格，按题型维度组织，支持Tesseract OCR兜底
- 自动化质检：答案自暴露检测、题干重复检测、选项长度均衡检测、解析质量检测、答案分布分析
- Word格式自动修复：公式转换、答案/解析样式修复、批量处理
- 三套OCR方案：PyMuPDF + Tesseract + RapidOCR，覆盖不同场景
- 断点续跑与分批控制：每批≤3卷，避免超时丢失

## 6. 未来方向
- 扩展更多省份和考类
- 教学场景的个性化适配
- 质检维度的持续丰富
- OCR精度的持续优化
- 出版社缩写映射扩展
