# 一课一练工具包 项目汇报 PPT（修订版）

## 设计原则
- 每页 ≤ 50 字正文
- 标题即观点（Action Title）
- 用图示/SVG代替文字
- 一页只讲一件事

## 页面规划（12页）

### Page 1: 封面
- 标题：一课一练试卷生成工具包
- 副标题：AI驱动的高职分类考试试卷批量生产系统

### Page 2: 核心问题
- 一句话：人工出题效率低，AI裸出质量差——我们需要一个可控的生产线
- 3个痛点卡片（每张≤15字）

### Page 3: 分层控制
- 标题：不只是让AI出题
- SVG图示：7层同心圆/堆叠架构图
- 底部点题一句

### Page 4: 生产流程
- 标题：从考纲到Word的标准化流水线
- SVG横版流程图：素材→规划表→生成→质检→交付
- 关键数据标注

### Page 5: 规划表
- 标题：一张表管住所有试卷
- 左侧SVG：规划表→控制生成器示意
- 右侧：3个关键数字

### Page 6: 真题风格
- 标题：让AI出的题像真题
- 左侧：真题→风格抽取→风格库示意
- 右侧：3类题型风格示例

### Page 7: 技术架构
- 标题：10个模块构建一条生产线
- SVG模块关系图（简化为5个核心模块）

### Page 8: OCR多方案
- 标题：三套OCR引擎覆盖所有场景
- 3列方案对比：RapidOCR / Tesseract / PyMuPDF

### Page 9: 质检体系
- 标题：6维质检保证交付质量
- 6个图标卡片（极简文字）

### Page 10: 覆盖成果
- 标题：已覆盖3省6考类
- 左侧数据卡片，右侧覆盖地图示意

### Page 11: 核心数据
- 标题：项目规模
- 4个KPI数字卡片

### Page 12: 结束页
