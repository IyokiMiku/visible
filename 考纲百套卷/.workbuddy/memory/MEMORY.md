# 考纲百套卷 项目记忆

## 项目概况
- 目标：为各省对口升学考试生成百套卷系列
- 已创建：重庆市电子信息类

## 目录结构（按新版说明）
- 规划表 + 细目表统一存放在 `04_生成输出/生产规划/{省份} {考类}/`

## 已完成
1. 重庆市电子信息类考点规划总表（94卷） + 映射表
   - 映射表：72个考点训练卷AI匹配→学科网kpointId, 14个专题训练卷+8个课程综合卷标"聚合"
   - 考纲：2027年重庆市电子信息类考试说明
   - 课程：电工技术基础与技能(40%)、电子技术基础与技能(40%)、单片机技术及应用(20%)
   - 原真题200分/120分钟 → 100分/60分钟(×0.5)
   - 题型：单选16题32分、判断16题32分、填空8题12分、综合4题24分
2. 细目表22个（14专题+8综合）
   - 每卷44题，8字段：题号/题型/难度/考查内容/对应考点/对应知识点/考纲要求/出题意图
   - 专题训练卷覆盖该专题全部考点
   - 课程综合卷多卷合并覆盖课程全部考点

## 关键约定
- 细目表命名：专题训练卷_第x卷_地区_专题名.docx / 课程综合卷_第x卷_地区_课程名.docx
- 难度分布：简单80%:适中10%:困难10%
- 掌握类知识点优先多出题
- 规划表编写说明已含第八章"唯一事实源与配置漂移防控"：xlsx最高事实源、planning_table.json运行时只读、cache+volXX派生禁止手工维护
- 配图风险前置检测系统已上线：5环节（规划扫描→config标记→出题规避→auto_review→QC优先），核心模块 image_risk.py，集成于 runner.py preflight + rules.py 质检
